################################################################################
#                           Sprightly - RMS 3                                  #
#                        CSS Stylesgeet Overview                               #
################################################################################

lib/ -> All reusable styles and plugins (in support of js libraries/plugins)
rms/ -> All RMS specific styles
images/ -> All images referenced by the CSS for look and feel usage only.


In detail:

################################################################################
### rms/rms_common.css

The core stylesheet for the entire RMS system.


################################################################################
### rms/rms_coolbox.css

Styles to support the dashboard widgets or "cool box" (see Work Tray widget on
the home page).


################################################################################
### rms/rms_ie6.css
### rms/rms_ie7.css

Style hacks to support display in IE.


################################################################################
### rms/rms_print.css

Stripped down styles for printing of RMS pages.


################################################################################
### rms/rms_search.css

Styles to support any page served by the SearchController i.e. the homepage,
search results and search widget.


################################################################################
### rms/rms_wizard.css

Styles to support the Agreement Wizard or any page served by the
WizardController.


################################################################################
### lib/dijit.css

Stylesheet to support the dojo toolkit - namley the dojo splitter container.

Source: http://www.dojotoolkit.org


################################################################################
### lib/jquery-contextmenu.css

Stylesheet to support the jquery-contextmenu.js plugin widget.

Source: http://abeautifulsite.net/notebook/80


################################################################################
### lib/jquery-multiselect.css

Stylesheet to support the jquery-multiselect.js plugin widget.

Source: http://abeautifulsite.net/notebook.php?article=62


################################################################################
### lib/jquery-tooltip.css

Stylesheet to support the jquery-tooltip.js plugin widget.

Source:  http://docs.jquery.com/Plugins/Tooltip


################################################################################
### lib/jquery-treeview.css

Stylesheet to support the jquery-treeview.js plugin widget.

Source:  http://bassistance.de/jquery-plugins/jquery-plugin-treeview/


################################################################################
### lib/jquery-ui.autocomplete.css

Stylesheet to support the jquery-ui.autocomplete.js plugin widget.

Source: http://docs.jquery.com/UI/Autocomplete


################################################################################
### lib/jquery-ui.css

Stylesheet to support the jquery-ui library of widgets, including the dialog,
accordion and tab widgets.

Source: http://jqueryui.com/
